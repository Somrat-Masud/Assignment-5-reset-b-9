What is the Document Object Model(DOM)?

The DOM allows the document's structure, style, and content to be manipulated
programmatically. Changes made to the DOM are immediately reflected in the displayed document.

How do you select an element from the DOM?

The select an element from the DOM, you use DOM selection methods provided by the document object in JavaScript.
These methods allow you to find elements by their tag, class, ID, attributes, or other selectors.

What is event delegation in the context the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events for its child elements.
